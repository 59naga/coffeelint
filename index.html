Installation

To install, make sure you have a working version of the latest stable version of Node and NPM (the Node Package Manager). Then you can run: npm install -g coffeelint Leave off the '-g' if you do not want to install globally.

Usage

Once you have Coffeelint installed, to lint your scripts, run: coffeelint application.coffee To specify your own configuration file, do the following: coffeelint -f config.json application.coffee

Options

CoffeeLint's rules: - ignore/error levels - everything is optional

Name Description
no_tabs
no_trailing_whitespace This options forbids trailing whitespace in your code. It is enabled by default.
max_line_length This rule imposes a maximum line length on your code.
indentation This rules imposes a standard number of spaces to be used for indentation. Since whitespace is significant in CoffeeScript, it's critical that a project chooses a standard indentation format and be consistent. The default number of spaces is set to two, though this is configurable.
camel_case_classes This rule mandates that all class names are camel cased. Camel casing class names is a convention in Javascript and Coffeescript, because it communicates that the `new` keyword is required when initializing. 
# Good!
class BoaConstrictor

# Bad!
class boaConstrictor
no_trailing_semicolons This rule prohibits trailing semicolons, since they are needless cruft in CoffeeScript. This rule is enabled by default. 
# This semicolon is meaningful.
x = '1234'; console.log(x)

# This semicolon is redundant.
alert('end of line')
no_implicit_braces This rule prohibits implicit braces when declaring object literals. Implicit braces can make code more difficult to understand, especially when used in combination with optional parenthesis. This rule is disabled by default, since it is not idiomatic CoffeeScript. 
# Some people may find this code ambiguous.
# Is it a function call with three arguments or four?
myFunction a, b, 1:2, 3:4

# While the same code written in an explicit manner has no ambiguity.
myFunction(a, b, {1:2, 3:4})
